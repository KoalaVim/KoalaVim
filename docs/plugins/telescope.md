# Telescope

Fuzzy finder and pickers.

| Plugin | Description |
| --- | --- |
| [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) | Fuzzy finder framework — files, buffers, LSP symbols, diagnostics, etc. |
| [telescope-fzf-native.nvim](https://github.com/nvim-telescope/telescope-fzf-native.nvim) | Native fzf sorter for telescope. |
| [telescope-ui-select.nvim](https://github.com/nvim-telescope/telescope-ui-select.nvim) | Route `vim.ui.select()` through telescope. |
| [telescope-live-grep-args.nvim](https://github.com/nvim-telescope/telescope-live-grep-args.nvim) | Live grep with raw rg args (globs, -t/-T filters). |
| [telescope-dict.nvim](https://code.sitosis.com/rudism/telescope-dict.nvim) | Dictionary/synonyms/spell suggestions picker. |
| [easypick.nvim](https://github.com/axkirillov/easypick.nvim) | Build custom telescope pickers from shell commands. |
