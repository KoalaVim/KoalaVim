# Libs

Shared libraries depended on by other plugins.

| Plugin | Description |
| --- | --- |
| [plenary.nvim](https://github.com/nvim-lua/plenary.nvim) | Lua utility library: async, path, job helpers used across the ecosystem. |
| [vim-repeat](https://github.com/tpope/vim-repeat) | Enables `.` repeat for plugin-provided operators (leap, surround, etc.). |
