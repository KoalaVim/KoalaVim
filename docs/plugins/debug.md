# Debug

Debug Adapter Protocol integrations.

| Plugin | Description |
| --- | --- |
| [nvim-dap](https://github.com/mfussenegger/nvim-dap) | DAP client — debug code from Neovim. |
| [nvim-dap-ui](https://github.com/rcarriga/nvim-dap-ui) | UI panels for nvim-dap (scopes, watches, breakpoints, stacks, repl). |
| [nvim-dap-virtual-text](https://github.com/theHamsta/nvim-dap-virtual-text) | Show variable values as virtual text inline while debugging. |
| [goto-breakpoints.nvim](https://github.com/ofirgall/goto-breakpoints.nvim) | Cycle between DAP breakpoints with jump-to mappings. |
