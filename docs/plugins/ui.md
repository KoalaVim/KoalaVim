# UI

Look & feel: statusline, bufferline, notifications, icons.

| Plugin | Description |
| --- | --- |
| [ofirkai.nvim](https://github.com/ofirgall/ofirkai.nvim) | Monokai-inspired color scheme. |
| [indent-blankline.nvim](https://github.com/lukas-reineke/indent-blankline.nvim) | Vertical indent guides. |
| [dressing.nvim](https://github.com/stevearc/dressing.nvim) | Prettier floating UIs for `vim.ui.input` and `vim.ui.select`. |
| [nvim-tree.lua](https://github.com/nvim-tree/nvim-tree.lua) | File explorer sidebar. |
| [lualine.nvim](https://github.com/nvim-lualine/lualine.nvim) | Fast and configurable statusline. |
| [git-blame.nvim](https://github.com/f-person/git-blame.nvim) | Git blame information in the statusline. |
| [nvim-navic](https://github.com/SmiteshP/nvim-navic) | LSP breadcrumb context for the winbar/statusline. |
| [bufferline.nvim](https://github.com/akinsho/bufferline.nvim) | Tabs/buffers line at the top. |
| [noice.nvim](https://github.com/folke/noice.nvim) | Nicer UI for messages, cmdline, notifications and LSP popups. |
| [fidget.nvim](https://github.com/j-hui/fidget.nvim) | Floating progress indicator for LSP/background tasks. |
| [colorful-winsep.nvim](https://github.com/nvim-zh/colorful-winsep.nvim) | Highlight the separators of the active window. |
| [statuscol.nvim](https://github.com/luukvbaal/statuscol.nvim) | Customizable status column (sign, number, fold columns). |
| [nvim-web-devicons](https://github.com/nvim-tree/nvim-web-devicons) | Filetype icons for other plugins. |
| [nui.nvim](https://github.com/MunifTanjim/nui.nvim) | UI component library used by other plugins. |
| [which-key.nvim](https://github.com/folke/which-key.nvim) | Popup showing possible keybinding continuations. |
| [alpha-nvim](https://github.com/KoalaVim/alpha-nvim) | Greeter/dashboard shown on startup. |
| [smear-cursor.nvim](https://github.com/sphamba/smear-cursor.nvim) | Smooth animated cursor trail. |
| [vimade](https://github.com/tadaa/vimade) | Dim inactive windows/buffers. |
