# KoalaVim Plugins

Plugins used by KoalaVim grouped by category. Each README lists the plugin name, a link to its upstream repository, and a short description.

| Category | Spec file | Description |
| --- | --- | --- |
| [AI](ai.md) | `plugins/ai.lua` | AI coding assistants and CLI agent integrations |
| [Autocomplete](autocomplete.md) | `plugins/autocomplete.lua` | Completion engine, sources and snippets |
| [Coding](coding.md) | `plugins/coding.lua` | Editing, motions, text objects, refactor helpers |
| [Debug](debug.md) | `plugins/debug.lua` | DAP debugger, UI and related helpers |
| [Editor](editor.md) | `plugins/editor.lua` | General editor enhancements and QoL utilities |
| [Games](games.md) | `plugins/games.lua` | Fun and learning plugins |
| [Git](git.md) | `plugins/git.lua` | Git integrations, diff viewers and forge helpers |
| [LSP](lsp.md) | `plugins/lsp/` | LSP client, servers, formatters and linters |
| [Sessions](sessions.md) | `plugins/sessions.lua` | Session save/restore |
| [Telescope](telescope.md) | `plugins/telescope.lua` | Fuzzy finder and pickers |
| [Terminal](terminal.md) | `plugins/terminal.lua` | Embedded terminal integrations |
| [Treesitter](treesitter.md) | `plugins/treesitter.lua` | Treesitter parsers, text objects and context |
| [UI](ui.md) | `plugins/ui.lua` | Look & feel: statusline, bufferline, notifications, icons |
| [Hydra](hydra.md) | `plugins/hydra/` | Modal sub-mode framework |
| [Libs](libs.md) | `plugins/libs.lua` | Shared libraries required by other plugins |
| [Koala](koala.md) | `plugins/koala.lua` | KoalaVim meta-plugin bootstrap |

See [`redundant-candidates.md`](redundant-candidates.md) for plugins that may be redundant in the AI-agent era.
