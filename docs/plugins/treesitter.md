# Treesitter

Treesitter parsers, text objects and context.

| Plugin | Description |
| --- | --- |
| [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) | Treesitter parser manager — syntax, folds, indent. |
| [nvim-treesitter-textobjects](https://github.com/nvim-treesitter/nvim-treesitter-textobjects) | Treesitter-powered text-objects (@function, @class, @block, @call). |
| [nvim-treesitter-context](https://github.com/nvim-treesitter/nvim-treesitter-context) | Sticky header showing the enclosing scope as you scroll. |
| [nvim-ts-context-commentstring](https://github.com/JoosepAlviste/nvim-ts-context-commentstring) | Context-aware commentstring (e.g. JSX inside TSX). |
| [jsonpath.nvim](https://github.com/phelipetls/jsonpath.nvim) | Show/yank JSONPath to the node under cursor. |
| [vim-matchup](https://github.com/andymass/vim-matchup) | Enhanced `%` matching for language constructs. |
| [nvim-treesitter-endwise](https://github.com/RRethy/nvim-treesitter-endwise) | Auto-insert matching `end` in Ruby/Lua/Bash/etc. |
