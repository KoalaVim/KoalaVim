# Autocomplete

Completion engine, sources and snippets.

| Plugin | Description |
| --- | --- |
| [nvim-cmp](https://github.com/hrsh7th/nvim-cmp) | Core completion engine for Neovim. |
| [cmp-nvim-lsp](https://github.com/hrsh7th/cmp-nvim-lsp) | LSP source for nvim-cmp. |
| [cmp-buffer](https://github.com/hrsh7th/cmp-buffer) | Current/visible buffer words source. |
| [cmp-path](https://github.com/hrsh7th/cmp-path) | Filesystem path completion source. |
| [cmp-cmdline](https://github.com/hrsh7th/cmp-cmdline) | Command-line (`:`) completion source. |
| [cmp-calc](https://github.com/hrsh7th/cmp-calc) | Inline arithmetic calculator source. |
| [cmp-look](https://github.com/octaltree/cmp-look) | Dictionary-based completion via `look`. |
| [cmp-git](https://github.com/petertriho/cmp-git) | Completion for commit messages (issues, PRs, users). |
| [cmp-snippy](https://github.com/dcampos/cmp-snippy) | Snippy source for nvim-cmp. |
| [cmp-lspkind-priority](https://github.com/ofirgall/cmp-lspkind-priority) | Sort completion entries by LSP kind priority. |
| [lspkind.nvim](https://github.com/onsails/lspkind.nvim) | VSCode-style pictograms for completion entries. |
| [nvim-snippy](https://github.com/dcampos/nvim-snippy) | TextMate/UltiSnips-compatible snippet engine. |
| [vim-snippets](https://github.com/honza/vim-snippets) | Community snippet collection. |
| [nvim-autopairs](https://github.com/windwp/nvim-autopairs) | Auto-insert matching pairs with treesitter awareness. |
| [crates.nvim](https://github.com/saecki/crates.nvim) | Cargo.toml completion, version info and management. |
