# LSP

LSP client, servers, formatters and linters.

## General

| Plugin | Description |
| --- | --- |
| [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig) | Collection of quickstart LSP server configurations. |
| [mason.nvim](https://github.com/mason-org/mason.nvim) | Package manager for LSP servers, DAP adapters, linters and formatters. |
| [mason-lspconfig.nvim](https://github.com/williamboman/mason-lspconfig.nvim) | Bridge between mason.nvim and nvim-lspconfig. |
| [none-ls.nvim](https://github.com/nvimtools/none-ls.nvim) | Bridge external formatters/linters/code actions to Neovim's LSP. |
| [conform.nvim](https://github.com/stevearc/conform.nvim) | Lightweight async formatter runner. |
| [lspsaga.nvim](https://github.com/glepnir/lspsaga.nvim) | Enhanced LSP UI: code actions, rename, lightbulb, diagnostics popups. |
| [inlay-hints.nvim](https://github.com/KoalaVim/inlay-hints.nvim) | Render LSP inlay hints as virtual text. |
| [format-on-leave.nvim](https://github.com/ofirgall/format-on-leave.nvim) | Format the buffer when leaving insert/normal mode. |
| [vim-illuminate](https://github.com/RRethy/vim-illuminate) | Highlight other occurrences of the word/symbol under the cursor. |
| [lsp_lines.nvim](https://github.com/ofirgall/lsp_lines.nvim) | Render LSP diagnostics as multi-line virtual text. |
| [diagflow.nvim](https://github.com/KoalaVim/diagflow.nvim) | Show LSP diagnostics as a floating widget. |
| [nvim-navbuddy](https://github.com/SmiteshP/nvim-navbuddy) | Interactive tree navigator for LSP document symbols. |
| [nvim-navic](https://github.com/SmiteshP/nvim-navic) | LSP breadcrumb context for the winbar/statusline. |
| [nvim-dr-lsp](https://github.com/ofirgall/nvim-dr-lsp) | Count of LSP definitions and references for the symbol under cursor. |
