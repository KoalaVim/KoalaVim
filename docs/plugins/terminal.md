# Terminal

Embedded terminal integrations.

| Plugin | Description |
| --- | --- |
| [toggleterm.nvim](https://github.com/akinsho/toggleterm.nvim) | Toggleable floating/split terminals with multi-instance support. |
| [term-edit.nvim](https://github.com/chomosuke/term-edit.nvim) | Vim-style editing at the shell prompt inside the built-in terminal. |
