# Coding

Editing, motions, text objects and refactor helpers.

| Plugin | Description |
| --- | --- |
| [Comment.nvim](https://github.com/numToStr/Comment.nvim) | Smart commenting with treesitter-aware commentstring. |
| [substitute.nvim](https://github.com/gbprod/substitute.nvim) | Operators to exchange and substitute text regions. |
| [nvim-autopairs](https://github.com/windwp/nvim-autopairs) | Auto-insert matching pairs with treesitter awareness. |
| [numb.nvim](https://github.com/nacro90/numb.nvim) | Peek at line jumps without leaving the current position. |
| [leap.nvim](https://github.com/ggandor/leap.nvim) | General-purpose motion plugin — jump anywhere visible. |
| [flit.nvim](https://github.com/ggandor/flit.nvim) | Enhances f/F/t/T with leap-style labels. |
| [nvim-toggler](https://github.com/nguyenvukhang/nvim-toggler) | Toggle opposite words (true↔false, prev↔next, etc.). |
| [yanky.nvim](https://github.com/gbprod/yanky.nvim) | Yank history ring — cycle through past yanks. |
| [nvim-surround](https://github.com/kylechui/nvim-surround) | Add/change/delete surrounding pairs via operators. |
| [treesj](https://github.com/Wansmer/treesj) | Split/join blocks of code using treesitter. |
| [sibling-swap.nvim](https://github.com/Wansmer/sibling-swap.nvim) | Swap treesitter siblings (arguments, list items). |
| [nvim-various-textobjs](https://github.com/chrisgrieser/nvim-various-textobjs) | Extra text-objects (subwords, indent, shell pipe, URL, etc.). |
| [mini.ai](https://github.com/echasnovski/mini.ai) | Enhanced a/i text-objects from mini.nvim. |
| [vim-visual-multi](https://github.com/mg979/vim-visual-multi) | Multi-cursor / multi-selection editing. |
| [move.nvim](https://github.com/fedepujol/move.nvim) | Move lines and blocks in any direction in visual mode. |
